# Assets locaux

## `hi4pi-density.png`

- source officielle : `HI4PI HI Column Density Map`
- origine : `https://lambda.gsfc.nasa.gov/product/foreground/f_images.html#HI4PI`
- fichier telecharge : `https://lambda.gsfc.nasa.gov/product/foreground/fg_images/med_HI4PI_image.png`
- usage dans Startrip : masque de densite angulaire pour la couche gaz du ciel local

Important :

- cette image represente une colonne integree de gaz neutre sur tout le ciel
- elle ne fournit pas de profondeur radiale par pixel
- dans le prototype, elle est extrudee en epaisseur pour creer une couche volumique jouable
